Getting started

An overview of APPCAKE Frontend Template, how to use, coding standard, and more.

Intro

APPCAKE Frontend Template based on the Bootstrap CSS style library development, response type structure.

HTML coding standard

Document statement

Template to use certain HTML elements and CSS properties need to set the page for the HTML document type. Each page in your project will be set with reference to the following format.

<!DOCTYPE html> <html lang="en"> ... </html>

Charset UTF-8

<meta charset="utf-8">

Mobile devices first

On the mobile device browser, through for the viewport set the meta attributes to the user-scalable=no can disable its scale (zooming) function. After such disable zoom function, users can scroll the screen, can make your site look more like native applications.

<meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=1, user-scalable=no">

SEO

  • title Website title, control within 25 words, 50 bytes. <title>...</title>
  • keywords Key words, about five, single 8 characters or less. <meta name="keywords" content="...">
  • description Site descriptions, try to empty words in 80 characters, 160 characters. <meta name="description" content="...">

ico

  • Bookmark The icon shown in the favorites. <link rel="Bookmark" href="...">
  • Shortcut Icon The icon shown in the address bar. <link rel="Shortcut Icon" href="...">

Basic template

<!DOCTYPE HTML> <html lang="en"> <head> <meta charset="utf-8"> <meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=1, user-scalable=no"> <title>...</title> <meta name="keywords" content="..."> <meta name="description" content="..."> <link rel="Bookmark" href="..." > <link rel="Shortcut Icon" href="..."> </head> <body> ... </body> </html>
