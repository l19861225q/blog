全局 CSS 样式

了解模版的全局 CSS 样式。

概览

深入了解模版结构的关键部分，让 Web 开发变得更好、更快、更强壮。

Normalize

为了增强跨浏览器表现的一致性，继承了 Bootstrap 的 Normalize Less 库（Normalize.css），这是由 Nicolas Gallagher 和 Jonathan Neal 维护的一个 CSS 重置样式库。

排版

页面主题

通用样式，默认主题：黑色背景，白色字体的效果。

body { color: #fff; font-family: "Helvetica Neue", Helvetica, "Hiragino Sans GB", "Microsoft YaHei", Arial, sans-serif; font-size: 62.5% line-height: 1.428; background-color: rgba(0,0,0,.7); }

反色主题：白色背景，黑色字体的效果，用于话题页等。通过为 body 元素添加 .-theme-inverse 类实现。

<body class="-theme-inverse"> ... </body> .-theme-inverse { color: #000; background-color: #fff; a, a:hover, a:visited, a:focus { color: #000; } }

辅助类

快速浮动

通过添加一个类，可以将任意元素向左或向右浮动。!important 被用来明确 CSS 样式的优先级。

<div class="pull-left">...</div> <div class="pull-right">...</div> .pull-left { float: left !important; } .pull-right { float: right !important; }

让内容块居中

为任意元素设置 display: block 属性并通过 margin 属性让其中的内容居中。

<div class="center-block">...</div> .center-block { display: block; margin-right: auto; margin-left: auto; }

清除浮动

通过为父元素添加 .clearfix 类可以很容易地清除浮动（float）。这里所使用的是 Nicolas Gallagher 创造的 micro clearfix 方式。

<div class="clearfix">...</div> .clearfix { &:before, &:after { content: " "; display: table; } &:after { clear: both; } }

图片响应式

通过给 img 添加 .img-responsive 类可以设置图片跟随父容器响应式的缩放尺寸。

<img class="img-responsive" src="图片地址" alt="图片描述"> .img-responsive { width: 100%; }

位置

通过添加 .ui-fixed-bottom 类可以设置元素固定在底部。

<div class="ui-fixed-bottom"></div> .ui-fixed-bottom { position: fixed; right: 0; left: 0; bottom: 0; }

间距

通过给模块添加 .mt-* 类可以设置各模块之间的垂直间距。（默认提供了 .mt-20 .mt-40 .mt-60 样式类，开发中可根据需要继续追加。）

<!-- <module-1>...</module-1> --> <!-- <module-2 class="mt-*">...</module-2> --> .mt-20 { margin-top: 20px; } .mt-40 { margin-top: 40px; } .mt-60 { margin-top: 60px; }

文章

通过为 p 添加 .-article 类，可以设置其排版属性。

<p class="-article">...</p> p.-article { font-size: 2.6em; line-height: 26px; text-align: justify; }
