本套模版基于 Bootstrap CSS 样式库开发，响应式结构。

HTML 书写规范

文档声明

模版中使用到的某些 HTML 元素和 CSS 属性需要将页面设置为 HTML5 文档类型。在你项目中的每个页面都要参照下面的格式进行设置。

<!DOCTYPE html> <html lang="zh-CN"> ... </html>

文档编码 UTF-8

<meta charset="utf-8">

移动设备优先

在移动设备浏览器上，通过为视口（viewport）设置 meta 属性为 user-scalable=no 可以禁用其缩放（zooming）功能。这样禁用缩放功能后，用户只能滚动屏幕，就能让你的网站看上去更像原生应用的感觉。

<meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=1, user-scalable=no">

SEO

title 网站标题，控制在25个字、50个字节以内 keywords 关键词，5个左右，单个8汉字以内 description 网站描述，字数尽量空制在80个汉字，160个字符以内

ico

Bookmark 收藏夹中显示的图标 Shortcut Icon 地址栏中显示的图标

基础模版

<!DOCTYPE HTML> <html lang="zh-CN"> <head> <meta charset="utf-8"> <meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=1, user-scalable=no"> <title>网站标题，控制在25个字、50个字节以内</title> <meta name="keywords" content="关键词，5个左右，单个8汉字以内"> <meta name="description" content="网站描述，字数尽量空制在80个汉字，160个字符以内"> <link rel="Bookmark" href="favicon.ico" > <link rel="Shortcut Icon" href="favicon.ico" /> </head> <body> ... </body> </html>

属性书写顺序

  • class
  • id, name
  • data-*
  • src, for, type, href, value
  • alt, title
  • style
  • other attrs

其他注意事项

注释

开始：<!-- .class-name --> 结束：<!-- /.class-name -->

img 添加 alt 属性，增加可访问性。

<img src="图片地址" alt="图片描述" title="图片描述">

CSS 书写规范

采用 RSCSS 结构

Reasonable Standard for CSS Stylesheet Structure（CSS样式表结构合理的标准）。

命名空间

自定义样式类统一采用 .ui- 开头。

属性书写顺序

  • 位置（Position）: position, top, right, bottom, left, float, clear, z-index ...
  • 盒模型（Model）: display, padding, margin, width, height ...
  • 排版（Typography）: color, font, line-xxx, text-xxx ...
  • 视觉（Visual）: background, border, box-shadow ...
  • 其他（Misc）: overflow, cursor, visibility ...

其他注意事项

注释

统一使用 /* ... */

上下模块之间的间距统一使用下一个模块的 margin-top 来实现，好处是：如果没有下一个模块也不会多出一段空隙。 （如果为列表元素中 item 设置 margin-top，需要设置负元素 margin-top 负值抵消）

链接 :link :visited :hover :active 书写顺序 L-V-H-A
